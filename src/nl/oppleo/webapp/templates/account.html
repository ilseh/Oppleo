Wat is 2-Factor Authenticatie

Met authenticatie in twee stappen of 2-Factor Authenticatie (2FA) wordt een account met zowel een wachtwoord (iets wat je weet) als een extra device zoals een telefoon (iets wat je hebt) beveiligd. Nadat 2-Factor Authenticatie is ingesteld wordt bij het inloggen om een wachtwoord Ã©n om een 6-cijferige code gevraagd. Deze code wordt gegenereerd door een app op de telefoon en heeft een geldigheid van ongeveer 30 seconden.

Om deze code te genereren is een app zoals Google Authenticator , Microsoft Authenticator of Lastpass Authenticator op een telefoon of tablet nodig.
